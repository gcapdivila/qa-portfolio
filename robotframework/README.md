# RobotFramework – Example Test Suite

This folder contains an example RobotFramework project using a Gherkin-like syntax.

## 🚀 Features

- Human-readable test cases
- Custom keywords (Python)
- Resource files for reusability
- Suitable for demonstrating keyword-driven automation

## 📁 Structure

```
robotframework/
├── tests/ # Test suites (.robot)
├── resources/ # Shared keywords / variables
└── keywords/ # Python keyword implementations
```


## ▶️ Run tests

```bash
robot tests/
```

## 📌 Notes

RobotFramework is particularly useful for demonstrating how QA teams collaborate with non-technical stakeholders.
