# API Testing – Postman / Newman

This folder contains an example of API test automation using Postman and Newman.

## 🚀 Features

- Postman collection (REST API tests)
- Environment variables
- Newman CLI runner
- HTML report generation (optional)
- CI/CD example (GitHub Actions)

## 📁 Structure

```
api-tests/
├── postman_collection.json
├── scripts/
│ └── newman-run.sh # Example Newman command
```


## ▶️ Run tests

```bash
newman run postman_collection.json
```

## 📌 Notes

This provides a clear demonstration of API-focused QA practices.
