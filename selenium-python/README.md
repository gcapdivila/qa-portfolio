# Selenium Python – Minimal Automation Setup

This folder contains a simple Selenium setup using Python + PyTest.

## 🚀 Features

- Clean and readable test structure
- Page Object Model (optional)
- Basic driver configuration
- Easy setup for demonstration purposes

## 📁 Structure

```
selenium-python/
├── tests/ # Test files
├── requirements.txt # Python dependencies
```


## ▶️ Run tests

```bash
pip install -r requirements.txt
pytest
```
