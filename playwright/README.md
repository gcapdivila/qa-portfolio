# Playwright Automation Framework

This folder contains a lightweight example of a Playwright-based UI automation framework.  
It demonstrates modern testing practices and a clean, maintainable structure.

## 🚀 Features

- Page Object Model (POM)
- Reusable fixtures & test configuration
- Robust waiting mechanisms
- Visual regression examples (`toHaveScreenshot`)
- API + UI hybrid tests
- Cross-browser execution (Chromium, Firefox, WebKit)
- CI/CD integration example (GitHub Actions)

## 📁 Structure (to be filled gradually)

```
playwright/
├── tests/ # Test suites
├── pages/ # Page Object Model classes
├── fixtures/ # Shared setup/config
├── utils/ # Helpers, data generators, selectors
└── package.json # Playwright dependencies
```

## ▶️ Run tests (placeholder)

```bash
npx playwright test
```
