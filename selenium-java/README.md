# Selenium Java – Example Automation Framework

This folder contains a minimal example of a Selenium-based UI automation framework using Java and Maven.

## 🚀 Features

- Page Object Model (POM)
- Reusable driver factory
- Basic test structure using JUnit/TestNG (TBD)
- Cross-browser capability (Chrome, Firefox)
- Simple, readable examples suitable for a QA portfolio

## 📁 Structure

```
selenium-java/
└── src/
├── main/java/ # Page objects, driver utils, config
├── test/java/ # Test classes
└── resources/ # Test data, configs
```


## ▶️ Run tests

```bash
mvn test
```

## 📌 Notes

This project is intentionally lightweight to keep the portfolio clear and focused.
